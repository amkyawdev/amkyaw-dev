🚀 Amkyaw.Dev — Modern Social Media Platform

Amkyaw.Dev is a modern social media web application built with a smooth and futuristic web architecture.

This platform combines real-time communication, social interaction, and immersive UI design to create a next-generation community experience.

The project focuses on smooth animations, modern layouts, voice chat groups, and real-time social networking features.

---

🌐 Project Vision

Amkyaw.Dev aims to build a modern social network platform inspired by systems like social communities, chat platforms, and voice interaction apps.

Users can:

- Share public posts
- Chat with friends
- Join voice chat groups
- Manage profiles
- Follow and interact with other users

The UI design emphasizes fluid interaction, smooth animation, and modern visual effects.

The pyramid seat layout represents active voice participants.

- Click seat to join
